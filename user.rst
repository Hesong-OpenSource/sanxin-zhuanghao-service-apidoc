用户管理
##############

用户管理API用于管理用户的账户信息和基本资料。

.. attention:: 在本系统中，用户使用手机号码登录，且专号与手机号码绑定，因此，手机号码被视作用户ID。

获取用户信息
================

.. http:get:: /api/user/(string: telenum)

  返回手机号码为 `telenum` 的用户的基本信息

  :>json string telenum: 用户的手机号码
